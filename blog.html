Discuss the scope of var, let, and const

var

var Variables declared with var have either global or function scope. 1.Global scope: A variable declared with var outside of any function is globally scoped. This means that it can be accessed from anywhere in your code.

2.Function scope: A variable declared with var inside of a function is function scoped. This means that it can be accessed from anywhere within that function, but not outside of it.

Let

Variables declared with let have block scope. This means that they can only be accessed from within the block of code in which they are declared.

Const

Variables declared with const also have block scope. Additionally, const variables cannot be reassigned.

Use cases of null & undefined

Use cases of null

To indicate that a variable does not have a value. For example, you might assign null to a variable that represents the current user if the user is not logged in.

To clear the value of a variable. For example, you might assign null to a variable that represents the selected item in a list when the user clicks on the clear

To represent the absence of a value in a database or other data store. For example, a database record might have a null value for the address field if the user has not entered their address yet.

Use cases of undefined

Use cases of undefined To indicate that a variable has not been assigned a value. For example, a variable might be undefined if it is declared in a function but never assigned a value.

To indicate that a function does not have a return value.

To represent the absence of a property on an object. For example, an object might not have a name property if it represents a new user who has not yet entered their name.

What is Rest API

REST API stands for Representational State Transfer API. It is a type of API that conforms to the REST architectural style. REST is a set of principles that define how data should be exchanged over the internet. REST APIs are typically designed to be used by other software applications, rather than by humans directly. They are often used to build web applications, mobile applications, and other types of software.
